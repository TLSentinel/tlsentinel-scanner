TLSentinel-scanner is the scanning agent for the TLSentinel certificate monitoring platform. It registers with the TLSentinel API using a scanner token, fetches the list of assigned hosts, performs TLS handshakes to collect certificate and cipher-suite data, and posts results back to the server. Scan concurrency and interval are configured server-side and picked up dynamically — interval changes take effect within one poll tick without restarting the agent. The scanner is designed to run alongside or remotely from the server, making it suitable for monitoring hosts across network boundaries.
